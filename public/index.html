FAQ

Does it send my regex or subject to a server?

No. PHP runs in your browser via WebAssembly. Network is used only to download static assets (the WASM runtime + library bundle).

Is the engine the same as PHP?

It’s PHP compiled to WebAssembly. Behavior is designed to match PHP’s PCRE2 engine closely, but always verify against your runtime for critical edge cases.

Can it help catch ReDoS vulnerabilities?

Yes. The ReDoS analyzer reports severity, a trigger explanation and suggested rewrites — with confidence levels.
